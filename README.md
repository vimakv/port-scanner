# port-scanner
A multithreaded TCP port scanner built in Python using socket programming. Supports custom port ranges, threading for fast scanning, and logs results of open, closed, and timeout ports.
# 🔍 Python TCP Port Scanner

A fast and simple **multi-threaded TCP port scanner** built using Python.  
This tool scans a target host to identify **open, closed, and filtered ports**.

---

## 🚀 Features

- ✅ TCP port scanning using Python sockets  
- ⚡ Multi-threading for faster scanning  
- 🎯 Custom port range support  
- 📄 Logging of scan results  
- ❗ Handles errors and timeouts  
- 💻 Command-line interface (CLI)

---

## 🛠️ Technologies Used

- Python
- Socket Programming
- Threading
- Queue
- Argparse
- Logging
