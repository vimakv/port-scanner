# 🔍 Python Port Scanner

A fast and simple **multi-threaded TCP port scanner** built using Python.  
It scans a target host to identify **open ports** and detect services.

---

## 🚀 Features

- ⚡ Multi-threaded scanning for speed  
- 🔍 Detect open ports  
- 🌐 Service detection (SSH, HTTP, etc.)  
- 🎯 Custom port range support  
- 📄 Logging support  
- 💻 Command-line interface (CLI)

---

## 🛠️ Technologies Used

- Python  
- Socket Programming  
- Threading  
- Queue  
- Argparse  
- Logging  
- Colorama
